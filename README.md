# 고립·은둔 청년 사례집 홍보 홈페이지

## 개요
고립·은둔 청년 23인의 사례를 담은 성과 사례집을 소개하는 정적 홍보 웹사이트입니다.  
당사자의 실제 이야기와 통계, 지원기관 정보, PDF 다운로드 링크를 제공합니다.

## 기술 스택
- HTML5 + CSS3 (Vanilla)
- JavaScript ES6+

## 로컬 실행 방법
1. VS Code에서 이 폴더를 엽니다.
2. Live Server 확장(예: `ritwickdey.LiveServer`)을 설치합니다.
3. `index.html` 파일을 우클릭 후 **"Open with Live Server"** 선택
4. 브라우저에서 `http://127.0.0.1:5500` (또는 표시되는 주소)로 접속합니다.

## 콘텐츠 수정 방법
- 사례 내용: `data/cases.json` 파일에서 카드 텍스트 수정
- 지원기관 정보: `index.html`의 `#support` 섹션 카드 내용 수정
- 색상/폰트: `style.css` 상단 `:root` CSS 변수를 수정
- PDF 파일: 프로젝트 루트에 위치한  
  `소이랩_고립은둔청년지원사업_성과사례집.pdf` 파일을 교체 (동일한 파일명 권장)

## 배포 방법 (GitHub Pages 예시)
1. GitHub에 새 저장소를 만듭니다.
2. 이 프로젝트의 모든 파일을 커밋 후 `main` 브랜치로 push 합니다.
3. GitHub 저장소의 **Settings → Pages** 메뉴로 이동합니다.
4. Source를 `main` / `/ (root)`로 설정합니다.
5. 저장 후 2~3분 뒤 제공되는 URL로 접속하면 사이트가 배포됩니다.

## 주의사항
- 사례 내용 수정 시 반드시 당사자 동의 여부를 확인해야 합니다.
- 사이트와 PDF는 비영리·교육 목적 활용을 권장하며, **상업적 이용은 금지**합니다.
