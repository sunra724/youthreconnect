## 🎨 1. 디자인 시스템 전면 수정 (Phase_01 반영)

### 색상 팔레트 변경
기존 "딥그린 계열" → **실제 사례집 "코랄/살구 계열"로 전면 교체**

```css
/* Phase_01에서 :root 변수를 아래로 교체할 것 */
:root {
  /* ✅ 실제 사례집 색상 (PDF 기반) */
  --color-primary: #E8765A;      /* 코랄/살구 - 제목, 강조, 버튼 */
  --color-primary-light: #F2A08A; /* 연한 코랄 - 호버, 배지 */
  --color-primary-pale: #FDE8E2;  /* 매우 연한 코랄 - 카드 배경 */
  --color-accent: #4A9B7F;        /* 민트그린 - 포인트 색상 (소량 사용) */
  
  /* 배경 */
  --color-bg-main: #F7F4F0;       /* 따뜻한 크림 베이지 - 메인 배경 */
  --color-bg-section: #FFFFFF;    /* 섹션 흰색 배경 */
  --color-bg-dark: #3D3530;       /* 다크 브라운 - 헤더, 다크 섹션 */
  
  /* 텍스트 */
  --color-text-main: #2C2420;     /* 다크 브라운 - 본문 */
  --color-text-sub: #7A6B63;      /* 미디엄 브라운 - 보조 텍스트 */
  --color-text-light: #F7F4F0;    /* 크림 - 다크 배경 위 텍스트 */
  
  /* 라인/보더 */
  --color-border: #E8DDD7;        /* 따뜻한 베이지 보더 */
  --color-line-accent: #E8765A;   /* 코랄 - 섹션 상단 라인 */
}
```

## 📄 2. PDF 다운로드 경로 수정 (Phase_04 반영)

### 실제 파일명으로 교체
```html
<!-- Phase_04의 다운로드 버튼에서 href 경로 교체 -->

<!-- ❌ 기존 (가상 경로) -->
<a href="./assets/사례집.pdf" download>PDF 다운로드</a>

<!-- ✅ 수정 (실제 파일명) -->
<a href="./소이랩_고립은둔청년지원사업_성과사례집.pdf" download="소이랩_고립은둔청년지원사업_성과사례집.pdf">
  PDF 다운로드 (약 17MB)
</a>
```

> **중요:** PDF 파일을 `index.html`과 **같은 폴더**에 복사해 두어야 합니다.
